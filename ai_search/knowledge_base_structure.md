# AI 챗봇 지식 기반 구조 가이드

이 문서는 AI 챗봇이 모의 면접 상황에서 '나'를 대신하여 답변할 수 있도록, 지식 기반(Knowledge Base)에 저장될 정보의 구조를 제안합니다. 이 구조를 기반으로 JSON 또는 Markdown 파일들을 생성하여 초기 지식 기반을 구축할 수 있습니다.

**목표:** 면접관의 질문에 대해 나의 경험과 역량을 담아 AI가 자연스럽고 상세하게 답변하도록 돕습니다. 각 정보 단위(스토리)는 면접 질문의 답변처럼 작성합니다.
